# Prodigy InfoTech Internship - Task 03

## 🎮 Tic-Tac-Toe Web Application

This project is part of my Web Development Internship at Prodigy InfoTech.

---

### 📌 Description
A simple and interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript. Two players can play the game by taking turns, and the application automatically detects the winner or a draw.

---

### 🚀 Features
- Two-player mode (X and O)
- Turn-based gameplay
- Win detection (rows, columns, diagonals)
- Draw detection
- Prevents clicking already filled cells
- Displays winner or draw message
- Restart game functionality
- Clean and responsive UI

---

### 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

### 🎯 How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
3. Start playing the game!
